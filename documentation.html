1. Theme Overview

NameTruePharm USA
Version1.0.0
AuthorTruePharm USA (placeholder — edit in style.css header)
Text Domaintruepharm
RequiresWordPress 6.5+ (tested 7.0) · WooCommerce 8.0+ (tested 10.8.1) · PHP 8.0+

A pure custom WooCommerce theme — no page builders, no CSS frameworks, no ACF. Custom data uses native register_post_type(), register_post_meta(), and add_meta_box().

Installation

  • Appearance → Themes → Add New → Upload Theme, choose truepharm-usa.zip, Install, Activate.
  • Settings → Permalinks → Save Changes (flushes rewrite rules).
  • Activate WooCommerce if needed.
  • Appearance → Menus → assign a menu to Primary Menu (slide-in panel).

Activation auto-creates

The after_switch_theme hook (truepharm_activate() in inc/theme-activation.php) runs once and:

  • Flushes rewrite rules (COA archive + rewards endpoint).
  • Creates tables {prefix}_tp_newsletter_emails & {prefix}_tp_contact_submissions.
  • Sets WC options: woocommerce_coming_soon=no, enable_myaccount_registration=yes, registration_generate_password=no, registration_generate_username=yes.
  • Schedules the daily tp_birthday_check cron.
  • Creates & template-assigns pages: why-us, faq, contact-us, rewards-program, legal, terms-of-use, privacy-policy, shipping-returns, compliance-statement.

Vial Size variants

Global attribute pa_vial_size (auto-created). Make the product Variable, add the Vial Size attribute & values, check Used for variations, then set a price per variation. The product page renders them as .variant-btn pills.

Redemption flow

ajax tp_redeem_points (logged-in) → validate multiple of 100 & balance ≥ → create WC_Coupon TP-REDEEM-{uid}-{ts} (fixed_cart, single-use, +30d, email-restricted) → deduct points → return {code, amount, balance, message}.

Referral & birthday

?ref=CODE → tp_rewards_capture_referral (init) sets the tp_referral_code cookie (30 days) → resolved on user_register. Birthday cron tp_birthday_check scheduled daily; guarded by tp_rewards_birthday_year.

13. Newsletter System

Table {prefix}_tp_newsletter_emails: id, email (unique), ip_address, source, created_at.

ajax tp_newsletter → tp_newsletter_subscribe (validates email, blocks dupes, fires tp_newsletter_subscribed).

SELECT email, created_at FROM wp_tp_newsletter_emails ORDER BY created_at DESC;

14. Contact Form

Table {prefix}_tp_contact_submissions: id, name, email, order_number, subject, message, submitted_at.

ajax tp_submit_contact → runs apply_filters('tp_contact_form_validate', new WP_Error()) (Turnstile), validates required fields, inserts the row, and emails get_option('admin_email') with Reply-To set to the sender.

SELECT * FROM wp_tp_contact_submissions ORDER BY submitted_at DESC;

15. Entrance Gate

Cookie: tp_compliance_agreed (TP_GATE_COOKIE), 30 days (TP_GATE_COOKIE_DAYS). Renders via tp_render_entrance_gate on wp_footer when the cookie is absent (not in admin).

ajax tp_set_gate_cookie (nonce tp_gate) sets the cookie. "Disagree" redirects to apply_filters('tp_gate_exit_url', 'https://www.google.com').

// Disable the gate entirely:
add_filter( 'tp_gate_should_show', '__return_false' );

18. Client Content Guide

  • Add a product — Products → Add New. Set a category (= Molecular Class), fill the Form field + Chemical Data / Storage tabs. For multiple sizes use a Variable product with the Vial Size attribute (price per size).
  • Add a COA — COA Library → Add New; title = compound name; fill Batch/Date/Purity; upload PDF; Publish.
  • Change top bar text — Appearance → Customize → TruePharm: Top Bar.
  • Set up Turnstile — Appearance → Customize → Cloudflare Turnstile → paste Site + Secret keys.
  • Newsletter subscribers — phpMyAdmin → wp_tp_newsletter_emails.
  • Contact submissions — phpMyAdmin → wp_tp_contact_submissions (also emailed to the admin).
  • Footer text / disclaimer — Appearance → Customize → TruePharm: Footer.
  • Legal pages — edit Terms of Use, Privacy Policy, Shipping & Returns, Compliance Statement in the normal editor.
